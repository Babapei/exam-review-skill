# Exam Review SOP (期末复习资料生成标准流程)

> 一个跨平台兼容的 AI Skill，用于将课程课件（PPT/PDF）+ 考试大纲自动整理成**适配具体题型**的期末复习资料包。
> 
> **关键特性**：不是固定输出7份文档，而是根据考试题型（问答/选择/编程/计算）和课程性质（理工科/文科/医学）**动态选择**文档组合。

---

## 这套 Skill 能做什么

上传你的 **所有课程资料**，AI 自动：

| 输入类型 | 示例 |
|---------|------|
| **考试大纲** | 分值分布表、题型说明、A/B卷安排 |
| **所有课件 PPT/PDF** | 两位老师的完整课件（本案例 12 份） |
| **老师录音/重点提示** | 最后一节课录音转写、"这个必考""这个不用看"的明确指令 |
| **历年真题/学长笔记** | 往届考试回忆、学长整理的大题复习题、手写笔记 |
| **补充说明截图** | 老师发的题型提示、"记不清了但大概是简答+计算" |
1. **识别考试题型**（从大纲关键词判断：问答题？选择题？编程题？计算题？）
2. **识别课程性质**（理工科公式型 vs 文科记忆型 vs 医学鉴别型）
3. **动态生成文档**（5-11份，不同题型组合完全不同）
4. **【v1.1 新增】** 若考试有明确的题型模式，额外生成一份**按题型组织的复习指南**：每题在卷子上长什么样→怎么答→练哪道→配哪里的知识

### 不同题型 → 不同输出

#### 问答题为主（如本次 NLP 考试）
| 文档 | 用途 |
|------|------|
| A4救命纸.md | 手写抄录：公式 + 对比表 |
| 期末复习指南.md | 完整知识点 |
| 考试问答预测.md | 30-50道预测问答 |
| 计算题精讲.md | 带步骤的计算例题 |
| 核心对比表.md | X vs Y 分析题专用 |
| 发展时间线.md | 记忆辅助 |
| 完整性评估报告.md | 查漏补缺 |

#### 选择题为主
| 文档 | 用途 |
|------|------|
| 知识点速记卡.md | 碎片化概念速记 |
| 易混辨析.md | 易错选项/陷阱分析 |
| 重点概念表.md | 一句话定义 |
| 期末复习指南.md | 完整知识点 |

#### 混合题型/算法类考试（v1.1 新增）

> 当考试有明确题型模式（简答+算法设计+计算+填空+选择），额外激活 Phase 6：

| 额外文档 | 用途 |
|---------|------|
| 01-全题型模版与解答指南.md | **卷子上每道题长什么样 + 怎么答 + 去哪练** |
| README 双计划导航 | Approach A（按章节） + Approach B（按题型） |

Phase 6 触发条件（自动检测，不需手动指定）：
- 考试有多个不同题型，且题干有重复模式
- 有历年真题或老师明确给出了出题格式
- 课程涉及算法/计算/过程类知识

**不触发**的考试类型：纯问答记忆型、纯选择题、题型完全不可预测的考试
| 文档 | 用途 |
|------|------|
| 代码模板.md | 常见算法模板 |
| 实验步骤.md | 实验流程 checklist |
| 调试清单.md | 常见错误修复 |
| 测试用例.md | 输入输出样例 |

## 安装方法（3种平台）

### 方法 1：OpenCode（推荐）

**方式 A：项目级**（只对当前项目生效）
```bash
# 复制 skill 到项目目录
mkdir -p .opencode/skills/exam-review-sop
cp SKILL.md .opencode/skills/exam-review-sop/

# 重启 OpenCode 生效
```

**方式 B：全局**（所有项目都能用）
```bash
mkdir -p ~/.config/opencode/skills/exam-review-sop
cp SKILL.md ~/.config/opencode/skills/exam-review-sop/
```

### 方法 2：Claude Code

```bash
mkdir -p ~/.claude/skills/exam-review-sop
cp SKILL.md ~/.claude/skills/exam-review-sop/
# Claude Code 会自动检测 ~/.claude/skills/ 下的 skill
```

### 方法 3：Claude Desktop (网页版)

1. 新建一个 **Project**
2. 在 Project Instructions 里粘贴 `SKILL.md` 的完整内容
3. 把课件和考试大纲上传到 Project files
4. 开始对话

### 方法 4：GitHub Copilot / Codex

创建 `.github/copilot-instructions.md`，把 `SKILL.md` 内容贴进去。Copilot 会自动读取。

## 使用方法

安装后，直接跟 AI 说：

> "帮我整理期末复习资料。这是考试大纲 [上传]，这是所有课件 [上传]，这是老师最后一节课录音转写 [上传]，这是往届真题和学长笔记 [上传]。考试是闭卷，混合题型。"

AI 会自动：
1. **检测题型** → 从大纲关键词判断是问答/选择/编程/计算
2. **检测课程性质** → 理工科/文科/医学/商科
3. **融合多来源资料** → 大纲定优先级 + 课件提取知识点 + 老师录音定取舍 + 真题验证出题模式
4. **选择文档组合** → 只生成对该题型有用的文档
5. **按6阶段执行** → 提取 → 整理 → 生成 → 检查 → 导出 → （可选）按题型优化

## 实际案例：本次 NLP 期末复习

这就是本 Skill 的**第一个实战案例**：

**输入**：
- 考试大纲：半开卷、100分、问答题、8章分值分布
- 8个PPT：578页，大量公式和架构图
- 时间：17小时突击

**检测**：
- 题型：问答题（关键词"问答题""理解""分析""应用"）
- 课程：理工科（CS/NLP，大量公式、模型架构）

**输出**：
- 生成了完整的7份问答题型文档
- A4纸公式可手写、计算题带步骤、50道预测问答
-  Markdown 公式使用 LaTeX 语法（`$...$` 包裹），Notion/VS Code/Typora 都能正确渲染
- 识别出10处遗漏并补充

这就是 Skill 中 "Essay + STEM" 模式的实际运行效果。

### 实际案例 2：算法设计与分析期末复习（v1.1 Phase 6 实战）

**输入**：
- 考试大纲 + 两位老师 12 份课件 + 老师录音转写 + 学长真题回忆/笔记
- 考试：闭卷、100分、混合题型（简答+算法设计+计算+填空+选择）
- 时间：2.5天突击

**检测**：
- 题型：混合（问答 + 算法设计 + 计算 + 填空 + 选择）
- Phase 6 触发：✅ 有 2024 真题模式可提炼 + 题型高度模式化

**Phase 1-5 输出**（知识导向，8份）：
- 期末复习指南、伪代码模板集、计算题精讲、考试问答预测、核心对比表、A4救命纸、2024大题复习、完整性评估报告

**Phase 6 输出**（题型导向，2份新增）：
- `01-全题型模版与解答指南.md`：17种题型，每题的卷子原样 + 答题步骤 + 练习题位置索引
- `00-考前冲刺导航.md`：8批任务链按分值排序（Master Method 3h → LCS 3h → Prim/Kruskal 3h → ...）
- README 提供双计划导航：Approach A 按章节 / Approach B 按题型

**效果**：2.5天从零到覆盖 100% 考点，用户直接按"这道题怎么拿满分"训练。

## 核心优势

1. **题型自适应**：问答题生成问答预测，选择题生成速记卡，编程课生成代码模板——不是一刀切
2. **考纲驱动**：按分值分配精力，不是按PPT页数
3. **跨平台**：OpenCode/Claude/Copilot 都能用
4. **可复用**：一次写好 skill，每学期换一批 PPT 直接跑
